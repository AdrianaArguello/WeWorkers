Se encarga de diseñar la lógica y las soluciones para que todas las acciones solicitadas en una página web sean ejecutadas de manera correcta.

Desarrollador frontend

Trabaja la interfaz desde el punto de vista del código para que el usuario pueda interactuar con un sistema.

Desarrollador Fullstack

Maneja todos los aspectos relacionados con la creación de una aplicación web. posee conocimientos en desarrollo Front-End y Back-End.
